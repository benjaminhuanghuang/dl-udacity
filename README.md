## Setup
```
$ sudo easy_install pip $ pip install --upgrade virtualenv

$ virtualenv --system-site-packages -p python3 venv3

$ . venv3/bin/activate 

(venv3) pip3 install -r requirements.txt

... (venv3)$ deactivate
```
